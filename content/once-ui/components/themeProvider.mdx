## SSR hydration

Inject this script into the `<head>` to set the initial theme before hydration:

<CodeBlock
  marginTop="16"
  marginBottom="24"
  codes={[
    {
      code:
`<script dangerouslySetInnerHTML={{ __html: \`
    (function() {
      function getInitialTheme() {
        try {
          const saved = localStorage.getItem("theme");
          if (saved === "system") {
            return window.matchMedia("(prefers-color-scheme: dark)").matches ? "dark" : "light";
          }
          return saved || "dark";
        } catch (e) {
          return "dark";
        }
      }
      document.documentElement.setAttribute("data-theme", getInitialTheme());
    })();
\` }} />`,
      language: "js",
      label: "Hydration script"
    }
  ]}
/>

## Behavior

When the theme is set, it updates the `data-theme` attribute on the `<html>` element.
