The `LogoCloud` component is used to display a collection of logos in a grid layout. It's perfect for showcasing partners, clients, or integrations on your website.

## Rotation

When you have more logos than the limit allows, the LogoCloud will automatically rotate through them.

<CodeBlock
  marginTop="16"
  marginBottom="24"

        { size: "m" },
        { size: "m" },
        { size: "m" },
        { size: "m" },
        { size: "m" },
        { size: "m" },
        { size: "m" }
      ]}
      limit={6}
