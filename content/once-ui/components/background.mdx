## Gradient Background

Create beautiful gradient backgrounds with customizable position, size, and colors.

<CodeBlock
  marginTop="16"
  previewPadding="0"
  marginBottom="24"
  preview={
    <Column fillWidth height={16}>
      <Background
        fill
        gradient={{ 
          display: true, 
          opacity: 1,
          x: 50,
          y: 0,
          colorStart: "brand-background-strong",
          colorEnd: "accent-background-weak"
        }}
      />
    </Column>
  }
  codes={[
    {
      code:
`<Background
  height="120px"
  fillWidth
  gradient={{ 
    display: true, 
    opacity: 0.8,
    x: 30,
    y: 50,
    colorStart: "var(--color-brand-medium)",
    colorEnd: "var(--color-success-medium)"
  }}
/>`,
      language: "tsx",
      label: "Gradient"
    }
  ]}
/>
